# Monster Mash
A game developed for game design class. Pet monster raising simulator. 3 stats to raise through 3 different minigames, win 4 competitions before losing 4 in order win.
Navigate the menus with arrow keys and enter
None of the minigames start until all the text menus are closed out
Speed minigame is controlled by left and right arrow keys. Player must dodge the boulders and earn speed based on how many boulders are dodged.
Stamina minigame is controlled by left, right, and up arrow keys. Player must avoid being hit by the arrows being shot, with the minigame ending after one hit.
Strength minigame is controlled by just clicking as fast as possible. Player must simply click as fast as possible for as long as possible, with strength gained based on how many clicks per second they could obtain.
The weekly competition at the end is a clean stat check between the player's monster and the 2 other monsters that were also training in the background, with some slight randomization.

Cheats:
G: Add 300 gold
A: Add 1 stamina
R: Add 1 strength
D: Add 1 speed
